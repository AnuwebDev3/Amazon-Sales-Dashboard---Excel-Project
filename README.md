# Amazon-Sales-Dashboard---Excel-Project
A beginner-friendly Excel dashboard project analyzing Amazon product data using pivot tables, charts, and visual insights such as ratings, reviews, and discount comparisons.
This Excel dashboard provides insights into:

  1. Top Rated Products
  2. Most Reviewed Products
  3. Top Discounted Products
  4. Discount vs Rating Comparison
  5. Summary pivot tables used to support the visualizations

These insights help identify product popularity, customer satisfaction, and best discount deals.
